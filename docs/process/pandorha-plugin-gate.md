# Gate Do Plugin Pandorha

## Decisao Atual

Criar plugin Pandorha esta bloqueado por enquanto.

## Criterios Para Reabrir

- Necessidade real de empacotar skills, MCPs e scripts para outro workspace.
- Configuracao MCP estavel e documentada.
- Instalacao reproduzivel em maquina limpa.
- Politica de versionamento para scripts, MCPs, skills e docs.

## Evidencia Necessaria

Antes de criar plugin, registre:

- quais capacidades precisam sair deste repo;
- quais comandos validam a instalacao;
- quais MCPs entram no pacote;
- quais skills sao incluidas ou excluidas;
- como rollback e upgrade serao feitos.

## Resultado Esperado

Enquanto o Pandorha Engine depender deste workspace local, mantenha scripts, MCPs e skills no proprio repositorio. Reavalie plugin apenas quando houver replicacao fora desta maquina ou necessidade clara de distribuicao.
