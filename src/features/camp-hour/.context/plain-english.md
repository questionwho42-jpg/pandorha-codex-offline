# Camp Hour In Plain Portuguese

Este modulo resolve uma hora simples de acampamento. Ele confere se cada personagem recebeu no maximo uma tarefa, registra o que cada um fez, aumenta o perigo e avanca o relogio de fortificar perimetro quando essa tarefa foi escolhida.

Ele ainda nao rola dados, nao cura personagens, nao gasta kits e nao cria encontros. A ideia e preparar uma base segura para a primeira tela de Acampamento.

Na primeira tela, o usuario escolhe uma acao para cada personagem, resolve 1 hora e ve o perigo, o relogio e o log. O estado resolvido ja entra no save local, entao pode ser salvo e carregado depois.
