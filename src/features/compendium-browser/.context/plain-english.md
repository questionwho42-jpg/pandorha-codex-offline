# Navegador Do Compêndio Em Linguagem Simples

A tela de Compêndio permite pesquisar regras já revisadas do Pandorha.

Nesta versão, ela usa uma lista pequena de entradas sobre criação de personagem, ancestralidades, classes e antecedentes. Ao clicar em uma entrada, o usuário vê um resumo e o arquivo de regra usado como fonte.

Ela ainda não lê todos os documentos do projeto e ainda não salva nada no banco.
