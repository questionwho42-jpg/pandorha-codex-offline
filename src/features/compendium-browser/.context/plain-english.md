# Navegador Do Compendio Em Linguagem Simples

A tela de Compendio permite pesquisar regras revisadas do Pandorha sem abrir os arquivos manualmente.

Nesta versao, ela usa entradas curadas e tambem um indice estatico gerado para regras de sobrevivencia, combate e magia. O usuario pode buscar por texto, filtrar por categoria e clicar em uma entrada para ver o resumo e a fonte no formato arquivo:linha.

Ela ainda nao le Markdown no navegador, nao interpreta mecanicas e nao salva nada no banco.

Em 2026-06-20, a tela ganhou filtros para entradas curadas, `Sistema: Sobrevivencia`, `Sistema: Combate` e `Sistema: Magia`.
