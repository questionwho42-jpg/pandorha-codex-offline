# Compendium Browser Scaling Roadmap

## Near Term

- Category filters for `system-survival`, `system-combat`, and `system-magic` are now delivered.
- Pagination, deterministic textual ranking, and contextual empty-state guidance are now delivered for the indexed browser.
- Preserve selected entry across query changes only after there is a stable detail route or state contract.
- Add direct page links or URL state only after the app has an approved routing/state persistence contract.

## Later

- Replace in-memory entries with Worker-backed queries after SQLite/OPFS is active.
- Add source-line deep links when source documents expose stable anchors.
- Add richer snippets after a validated Markdown import pipeline exists.
- Add semantic search only after a separate evidence and governance gate approves embeddings/AI behavior for discovery metadata.
