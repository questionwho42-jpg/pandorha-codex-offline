# Inventory Readonly Para Usuario

Esta tela mostra uma mochila de treino com itens fixos e calcula se a carga esta normal, lenta ou imobilizada.

Por enquanto o usuario nao compra, remove ou equipa itens. A tela serve para confirmar que a regra de slots ja funciona de forma visivel no navegador.

O proximo fluxo aprovado substituira esta carga fixa por itens carregados por
personagem, depois que o ledger e o save v6 estiverem prontos.
