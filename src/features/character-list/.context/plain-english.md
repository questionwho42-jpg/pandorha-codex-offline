# Character List In Plain English

The character list is the first real screen inside the `Personagens` tab.

Right now it shows characters created during the current browser session. It does not load permanent saves yet.

It now shows ancestry, class, and background as readable Portuguese names when the app provides the official catalogs.

This keeps the browser progress visible while the database and creation flow are still being built in small, reviewable steps.

The list now also shows the three ancestry traits saved for each character. It displays the trait name and description from the catalog when possible.

Those traits are still read-only notes on the sheet. They do not change numbers, HP, attacks, defenses, or starting equipment yet.
