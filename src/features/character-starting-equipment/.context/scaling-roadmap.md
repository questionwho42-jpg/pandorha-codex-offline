# Character Starting Equipment Scaling Roadmap

- Add user choice between official kit and alternate purchases only after an economy/gold contract is approved.
- Add automatic equip suggestions only after every kit item has a structured loadout/combat profile.
- Keep backfill for existing saves out of this feature unless a migration or explicit repair flow is approved.
- Do not add save v9 for kit grants; current ownership is represented by `inventoryEvents`.
- Future class variants should extend the resolver with explicit tests for every class id and catalog id.
