# Background Plain English

This module is the technical catalog of the official Pandorha backgrounds.

It answers simple questions such as:

- What backgrounds exist today?
- What is each background called for the player?
- Which rule file did the background come from?
- What origin ability and starting talent options are associated with it?

The code does not create characters, save anything to the database, apply starting resources, calculate bonuses, or activate talent powers yet. It only validates that the official background records are well-formed and can be listed or found by id.

The visible names stay in Portuguese for the user, while the internal ids stay in English for code stability.
