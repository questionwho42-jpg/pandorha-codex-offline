# Character Class Plain English

This module is the technical catalog of the four base Pandorha classes.

It answers simple questions such as:

- What classes exist today?
- What is each class called for the player?
- Which rule file did the class come from?
- What are its base HP, equipment text, passive ability, and initial talent options?

The code does not create characters, save anything to the database, calculate final stats, or apply class powers yet. It only validates that the official class records are well-formed and can be listed or found by id.

The visible names stay in Portuguese for the user, while the internal ids stay in English for code stability.
