# Dialogue Choice Para Usuario

Este modulo guarda as primeiras escolhas de argumento para negociacao social.

Por enquanto existem tres opcoes de treino: persuadir, barganhar e pressionar. Cada uma tem um texto visivel e um pequeno modificador que uma tela futura podera aplicar ao apelo social.

A alternativa seria escrever essas opcoes direto na tela de Relacoes, mas isso dificultaria testar e reutilizar as escolhas em dialogos futuros.
