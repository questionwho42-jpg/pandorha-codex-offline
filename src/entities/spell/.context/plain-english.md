# Spell Para Usuario

Este modulo guarda a lista basica de magias conhecidas pelo motor. Ele registra nome, circulo, custo em EE, componentes, se a magia usa ataque ou teste de resistencia, texto de dano e fonte da regra.

Na T26, isso ainda nao permite conjurar magia. A alternativa seria criar conjuracao junto com o catalogo, mas isso misturaria regras demais. A escolha atual deixa primeiro os dados confiaveis para a proxima etapa montar o fluxo de conjuracao.
