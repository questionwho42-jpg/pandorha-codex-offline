# Compêndio Em Linguagem Simples

O módulo de compêndio é o primeiro índice interno das regras do Pandorha.

Nesta etapa ele não lê todos os documentos automaticamente. Ele guarda uma lista pequena e revisada de entradas importantes para criação de personagem: guia de ficha, ancestralidades, classes e antecedentes.

Isso prepara a próxima tela de Compêndio para pesquisar regras sem depender de texto bruto espalhado em muitos arquivos.

Na T17A, esse módulo também ganhou uma busca simples. Ela encontra termos mesmo quando o usuário digita com maiúsculas diferentes ou sem acentos.
