# Ancestry In Plain English

The ancestry catalog lists the six basic origins a Pandorha character can come from.

For now, it stores the visible name, the official source document, the starting bonus text, and the primordial ability text. It does not yet let the player choose ancestry traits or apply bonuses automatically.

This is a foundation step: future tasks can connect this catalog to the character creator and then add the three trait choices from each ancestry.

T12 added the list of selectable traits. Each ancestry now has 10 official trait options, and the service can check whether a level-one character picked exactly 3 valid traits from the chosen ancestry.

The game still does not apply the trait effects automatically. At this point, the traits are reliable text records that future screens and rules can use.
