# Npc Para Usuario

Este modulo guarda uma pequena lista de NPCs de treino para o jogo usar depois em cenas sociais.

Ele ainda nao cria dialogo, conversa interativa, recrutamento ou save novo. Por enquanto ele so responde perguntas simples como: "quais NPCs existem?", "qual NPC tem este id?" e "quais NPCs pertencem a esta faccao?".

Alternativa mais simples seria colocar esses NPCs diretamente na tela de Relacoes, mas isso misturaria dados de regra com interface. A opcao atual e melhor para o projeto porque deixa a proxima etapa de negociacao social testar o catalogo sem depender da UI.
